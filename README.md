Izobraževalna igra Radiokarbonska metoda, razvita v okviru predmeta Interaktivnost in oblikovanje informacij, Fakulteta za racunalništvo in informatiko, UL
Avtorici: Tjaša Jagodnik, Viki Petrovic

Igra je razvita z uporabo tehnologij HTML, CSS in Javascript. Vse zahtevane datoteke, slike in ostali viri so že vkljuceni.

Igro se namesti (kopiranje vseh datotek) kot spletno aplikacijo, torej v root mapo uporabljenega spletnega strežnika, ki sam prepozna datoteko index.html kot zacetno stran. 


Igra je objavljena tudi na strežniku Heroku, na naslovu: https://radiocarbon.herokuapp.com
